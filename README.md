Este projeto consiste no desenvolvimento de um site interativo de receitas inspirado no personagem Chico Bento,
utilizando as tecnologias HTML, CSS e JavaScript. A proposta da atividade foi criar uma página web temática que
apresentasse receitas típicas da roça de forma visual, animada e divertida para o usuário.

Ao acessar o site, o usuário encontra um ambiente com temática rural, utilizando um fundo de madeira e cores que remetem 
ao universo do Chico Bento. Na parte superior da página existe um menu de navegação com as opções Sobre, Salgados, Doces 
e Contato. Essas opções permitem ao usuário visualizar informações sobre o site, filtrar receitas por categoria ou acessar 
informações de contato.

No centro da página há um botão chamado “Ver Todas as Receitas”. Quando esse botão é clicado, um chapéu do Chico Bento que 
aparece inicialmente na tela desaparece com uma animação, e as receitas passam a ser exibidas em forma de cartões com imagens
e nomes dos pratos. As receitas estão divididas em duas categorias principais: salgados e doces.

Cada receita possui uma imagem clicável. Quando o usuário clica em uma delas, uma janela modal aparece na tela mostrando
o nome da receita, os ingredientes e o modo de preparo. Essa janela pode ser fechada clicando no botão de fechar.

O site também possui diversas animações e interações feitas com JavaScript para tornar a experiência mais dinâmica. 
O cursor padrão do mouse foi substituído por um cursor personalizado em formato de garfo. Existe também um personagem do 
Chico Bento que atravessa a tela continuamente, e quando o usuário clica nele várias vezes, ele reage como se estivesse 
sendo “comido”, diminuindo de tamanho e desaparecendo temporariamente antes de voltar.

Outro elemento interativo é um personagem ajudante que fica no canto da tela e muda de posição quando clicado.
Além disso, há uma cesta com alimentos representados por emojis. Quando o usuário passa o cursor sobre esses alimentos, 
eles desaparecem como se estivessem sendo comidos e novos alimentos aparecem automaticamente.

O botão principal do site possui uma animação de cores em movimento criada com CSS, e os cartões de receitas possuem
efeitos de zoom quando o usuário passa o mouse sobre as imagens.

O objetivo dessa atividade foi praticar a construção de páginas web utilizando HTML para estruturar o conteúdo,
CSS para estilizar e criar animações visuais, e JavaScript para adicionar interatividade e manipulação de elementos
da página. O resultado final é um site temático de receitas que combina programação, design e animações para criar uma
experiência divertida para o usuário.
